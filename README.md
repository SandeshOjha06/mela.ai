# mela.ai: Autonomous Event Command Center

mela.ai is a scalable, multi-tenant application for managing large-scale event logistics with AI-powered autonomous agents. Every operation is scoped by event ID for complete data isolation. The platform leverages a swarm of specialized LangGraph agents to handle event tasks such as marketing, scheduling, participant communication, and emergency response.

## Architecture

The system is built on a modern stack:
- **Backend**: FastAPI (Python), PostgreSQL, ChromaDB (for RAG), and LangGraph for the AI agent swarm.
- **Frontend**: React Single Page Application (SPA), styled with Tailwind CSS, providing dashboards for both organizers and participants.
- **Agents**: Specialized AI agents (Problem Solver, Marketing, Scheduler, Email, Emergency Info, Budget/Finance) orchestrate all incoming requests using a Star Topology pattern overseen by a Supervisor agent.

## Components

### Backend
The backend provides APIs for event organizers to:
- Create and manage events
- Generate join codes for participants
- Trigger the AI swarm with instructions
- Handle HITL (Human-in-the-Loop) resolution for complex queries
- Manage priority queues for participant support tickets

### Frontend
The frontend features:
- **Organizer Dashboard**: A command center to oversee event status, view AI agent interaction logs, address unresolved queries, and manage event schedules.
- **Participant Portal**: A dedicated portal where attendees can ask questions, view schedules, and receive live updates generated by the AI swarm.

## Setup and Installation

### Prerequisites
- Python 3.12+
- Node.js 18+
- Docker and Docker Compose (optional, for containerized deployments)
- PostgreSQL database
- API Keys (Groq, etc.)

### Backend Setup
1. Navigate to the `backend` directory.
2. Install dependencies (e.g., using `pip install -r requirements.txt` or equivalent).
3. Copy `.env.example` to `.env` and fill in your configuration (Database URL, API keys).
4. Run the FastAPI development server:
   ```bash
   uvicorn main:app --reload
   ```

### Frontend Setup
1. Navigate to the `frontend` directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the Vite development server:
   ```bash
   npm run dev
   ```

## Docker Deployment

The application is fully containerized. To run the entire stack using Docker Compose:
```bash
docker-compose up --build
```
This will spin up both the frontend and backend services along with any required infrastructure.
